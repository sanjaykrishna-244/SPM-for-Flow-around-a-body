# SPM-for-Flow-around-a-body
Source Panel Method to model flow around a non-lifting semi-infinte body approximated to 2d flow problem using potential flow. 
The code is written using python, numpy, pandas and matplotlib libraries. 
# For Clyindrical Geometry
It captures the flow velocities on the surface of the cylinder and the pressure coefficient. It also plots the same varying with the polar angle. The panel geometries is defined by the parameter 'n'. The geometry and the influence matrices along with source strengths and other important values get saved in an excel file.
Finally, a streamplot is plotted showing the stream lines of the flow around the cylinder.
